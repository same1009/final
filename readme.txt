Summary: 
This application lets users log into a community where they can upload recipes, give reviews and rating other people's recipes, and search for recipes. 

Below are the following gems that are used:
devise
paperclip
will_paginate

Available functionality:
a) Users can sign up and log in
b) Users can create/edit/destroy their recipes
c) Users can review other user's recipes
d) Users can search for recipes to review/rating
e) When you view a recipe, it also lists all of the recipe's reviews. 

Models in App:

User
Recipe
Review
